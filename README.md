# Paquete Stata: GINITWF
GINI VARIABLES DISCRETAS


Paquete para cálculo de coeficientes de Gini educativos basado en:
1. Deaton, A. (1997) - Para variables continuas
2. Thomas, Wang & Fan (2001) - Para variables discretas (Gini Educativo)

## Instalación

```stata
* Instalar desde GitHub:
net install ginitwf, from("https://raw.githubusercontent.com/washingtonquintero/ginitwf/main/") replace

* O instalar localmente:
net install ginitwf, from("C:\ruta\local\") replace
